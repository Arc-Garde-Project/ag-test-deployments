Pattern 07

Grid-Rows Accordion

grid-template-rows: 0fr to 1fr with transition. Smoother than the max-height hack. No arbitrary values. Native CSS collapse.

grid-template-rows: 0fr/1fr + overflow: hidden Looks 6/10 | Actually 1/10
Three absolutely-positioned divs with massive box-shadow blur radii (45vmax each). Each animated on position with staggered durations (9s, 14s, 21s) and hue-rotate cycling. The blur creates soft blending. Pioneered by Linear, now the defining dark-mode SaaS aesthetic.
A ::before pseudo-element uses conic-gradient(from var(--angle)) that rotates via @property --angle animation. The mask-composite: exclude technique constrains the gradient to appear only as a 1px border. The ::after pseudo covers the card interior. On hover, the ::before fades in with opacity transition.
A radial-gradient positioned at CSS custom properties --spot-x and --spot-y. JavaScript mousemove listener calculates cursor position relative to the card and sets these properties. The overlay has opacity: 0 by default, transitioning to opacity: 1 on hover. The gradient creates a soft glow circle that follows the cursor.
The logos are duplicated in the HTML (two identical sets). The container uses width: max-content with animation: translateX(-50%). When the animation reaches the duplicate set, it loops seamlessly because both sets are identical. Edge fading uses mask-image: linear-gradient() on the wrapper. Pauses on hover with animation-play-state: paused.
Instead of the classic max-height: 0 to max-height: 999px hack (which has timing issues), this uses grid-template-rows: 0fr transitioning to 1fr. The inner content has overflow: hidden. This creates a perfectly smooth collapse/expand with no arbitrary values. The browser calculates the exact height. Works in all modern browsers.
