Enterprise Product Graphics

7 complete patterns showing how SaaS companies turn their product UI into marketing graphics. Each file is a standalone HTML document with all CSS/JS inline.

Pattern 01
Floating UI Vignettes
Fake product interfaces floating on a dark background with depth shadows, 3D perspective, and ambient glow. Multiple panels at different translateZ depths.
Used by: Stripe, Linear, Raycast
Pattern 02
Product Flow Graphics
Multiple UI panels connected by animated SVG lines showing a workflow. Code editor to build step to deployment to live site. Staircase layout with marching ants and flowing dots.
Used by: Vercel, GitHub, Retool
Pattern 03
Animated Code Blocks
Code that types itself character by character with syntax highlighting, blinking cursor, and streaming terminal output. Token-based typing engine with 6 color categories.
Used by: Stripe, Raycast, Supabase
Pattern 04
Product State Transitions
UI elements that change state with smooth animations. Tasks completing and reordering, number counters rolling, toggle switches sliding, progress bars filling.
Used by: Linear, Notion, Todoist
Pattern 05
Bento Product Grids
Multiple product feature vignettes in a bento grid layout. Each cell shows a different feature with its own animation. CSS Grid with named template areas and mixed cell sizes.
Used by: Apple, Notion, Arc Browser
Pattern 06
Browser & Device Frames
CSS-only browser, phone, and laptop frames wrapping live HTML content. Address bar, traffic lights, Dynamic Island, keyboard base. Three complete device templates.
Used by: Webflow, Framer, Linear
Pattern 07
Connection Lines & Data Flow
SVG lines connecting UI panels with 5 animation variants: marching ants, flowing dots, pulse waves, drawing lines, and glowing paths. Dynamic JavaScript positioning for CSS-layout elements.
Used by: Stripe, Retool, Vercel

Quick Reference: Key CSS Properties

Effect Properties Values
3D Space perspective 1000px - 1500px on parent container
Depth Layers transform-style: preserve-3d + translateZ() preserve-3d on group, 0-60px translateZ on children
Float Tilt transform: rotateX() rotateY() 5-12deg for subtle, 15-25deg for dramatic
Layered Shadows box-shadow (4-5 layers stacked) 1px/4px/16px/32px blur radius, decreasing opacity
Ambient Glow radial-gradient on ::before Accent color at 10-15% opacity, 600px wide
Marching Ants stroke-dasharray + stroke-dashoffset "8 5" dasharray, animate offset to -(dash+gap)
Path Animation SVG <animateMotion> dur: 2-4s, repeatCount: indefinite
Drawing Effect stroke-dasharray = stroke-dashoffset = pathLength Animate dashoffset to 0
Glow Filter SVG <feGaussianBlur> stdDeviation: 2-4 for subtle, 6-8 for strong
Number Counter requestAnimationFrame + easing 1.5-2.5s duration, easeOutCubic
Toggle Slide transform: translateX() Thumb slides by (track-width - thumb-width - padding)
Cursor Blink animation: steps(1) 0.9s duration, alternating opacity 1/0
Premium Easing cubic-bezier(0.23, 1, 0.32, 1) The "Linear/Stripe" deceleration curve
Bento Grid grid-template-areas Named areas with consistent gap (12-16px) and radius (16-20px)
Tabular Numbers font-variant-numeric: tabular-nums Prevents number width shift during counting
